Lista comandi chat gpt

Oct 13, 20259 min read

comandi utilizzabili per il compito: append(L1,L2,L3) → unisce due liste: L3 = L1 + L2. atom(A) → vero se A è un atomo (es. a, pippo). number(A) → vero se A è un numero. member(Elem,L) → vero se Elem appartiene alla lista L. list(L) → vero se L è una lista. reverse(L,Lrev) → vero se Lrev è la lista L al contrario. length(L,N) → vero se N è la lunghezza di L.
